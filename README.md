# World Happiness Report Dashboard (2015–2019)

## Overview
An interactive Power BI dashboard analysing the World Happiness Report 
dataset across 156 countries from 2015 to 2019.

## Dashboard Pages
- **Page 1 — Map View:** Global happiness scores visualised on a world map
- **Page 2 — Country Rankings:** Top 10 happiest and bottom 10 unhappiest countries
- **Page 3 — Trends Over Time:** Happiness score trends across 5 years
- **Page 4 — What Drives Happiness:** Correlation analysis of 6 key factors
- **Page 5 — Key Insights:** Summary findings and conclusions

## Key Findings
1. GDP per capita, social support and health are the strongest predictors of happiness
2. Finland consistently ranks as the world's happiest country from 2018 onwards
3. Global average happiness score remained stable at approximately 5.38
4. Generosity and trust in government show weaker correlations with happiness

## Tools Used
- Microsoft Power BI Desktop
- Power Query for data cleaning and transformation
- DAX for custom measures
- Data source: World Happiness Report (Kaggle)
